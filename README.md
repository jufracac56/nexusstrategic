# Landing Page SITA

## Despliegue en GitHub Pages

1. Crear un repositorio para la landing (ej: `sita-landing`).
2. Subir el contenido de esta carpeta.
3. En GitHub: Settings → Pages → Deploy from branch → `main` `/root`.
4. Editar el archivo `CNAME` con el subdominio real (ej: `sita.tudominio.gob.mx`).
5. En Route53, crear un registro CNAME apuntando a `tuusuario.github.io`.

## Archivos
- `index.html`
- `styles.css`
- `script.js`
- `CNAME`
